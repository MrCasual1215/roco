1. `00_project_description.md`：项目是什么；
2. `01_what_we_need_to_do.md`：我们要做什么；
3. `02_runbook.md`：怎么运行；
4. `03_code_architecture.md`：代码结构；
5. `04_model_and_prompt.md`：模型接入与 prompt；
6. `05_task_notes.md`：各任务策略；
7. `06_experiment_log.md`：实验记录；
8. `07_failure_analysis.md`：失败分析；
9. `08_todo_and_ownership.md`：分工与待办；
10. `09_submission_checklist.md`：提交前检查。
